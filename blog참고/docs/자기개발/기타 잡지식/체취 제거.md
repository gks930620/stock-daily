# 체취·입냄새·땀 관리 정리

## 1) 몸에서 냄새가 날 때

- 샤워 전, 마른 상태에서 귀 뒤/뒷목/겨드랑이/배꼽/사타구니 등 노폐물 부위를 관리
- 클렌징오일을 바르고 문지른 뒤, 물을 소량 묻혀 유화 후 샤워
- 주 1~2회 관리가 체취 감소에 도움 될 수 있다는 메모

## 2) 입냄새가 걱정될 때

- 혀 안쪽까지 닦는 것이 중요
- 작은 칫솔모, 혀 클리너 활용 가능
- 체크 방법: 혀 안쪽에 손가락을 3~5초 대고 냄새 확인
- 보조 수단: 미니 가글/구취 스프레이/캡슐형 구취 제품

## 3) 체취보다 땀이 더 걱정될 때

- 롤온/스프레이형 데오드란트 활용
- 쿨링 스프레이는 몸이 아니라 옷에 사용
- 외출 시 티슈·손수건, 얼굴은 기름종이 활용
- 겨드랑이 땀이 많으면 티셔츠 안쪽 패드 사용

## 4) 잘 씻는데도 냄새가 날 때

- 원인이 몸보다 옷 관리일 수 있음
- 섬유유연제 과다 사용 주의
- 건조 불충분 시 냄새가 남기 쉬움
- 햇볕 건조/건조기/실내건조용 세제 활용
- 냄새 위에 탈취제를 덮어 쓰면 악화될 수 있음

## 5) 냄새를 악화시키는 행동

- 향수/탈취제를 과하게 덮어 쓰기
- 땀을 닦지 않고 그대로 두기

##  요약
몸 냄새는 노폐물 관리,  
입냄새는 혀 청소,  
땀은 바로 닦기,  
옷 냄새는 세탁/건조 습관 점검,  
향으로 가리기보다 원인 해결이 우선.
