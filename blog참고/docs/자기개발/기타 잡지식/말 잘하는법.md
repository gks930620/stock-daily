# 말 잘하는 법 (논리적으로 말하기)

1. **말할 주제를 분명히** 한다.
2. **순차 배열법** — 범주에 맞춰 순서대로 말한다.
3. **단문으로** 말한다. — "~인데", "~해서"로 길게 이으면 상대가 정보 처리하기 어렵다.
4. **구어체보다 문어체식으로.** — 글도 단문으로 쓰는 연습을 한다.
5. **쌍괄식으로** 말한다. — **주장 먼저 → 근거 → 주장 한 번 더.**

> 핵심: 짧게(단문), 순서 있게(범주), 그리고 **결론을 앞뒤로 감싸서**(쌍괄식) 말한다.
