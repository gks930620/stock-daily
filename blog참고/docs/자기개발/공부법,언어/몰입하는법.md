# 몰입과 집중, 그리고 휴식

## 1. 진짜 휴식이란

- **집중해야 하는 활동은 휴식이 아니다.** (유튜브·책 모두 집중이 필요하므로 휴식 X)
- **집중을 안 하는 것**이 휴식이다 → 시각적으로 집중 안 해도 되는 것: **눈 감기, 산책, 명상**.

## 2. 몰입 환경 만들기

- 방해되는 것은 **치운다.**
- 특히 **시각적 방해 요소 제거**가 중요하다.
- 너무 조용한 곳보다 **카페·도서관 정도의 백색소음**이 집중에 최고다.

## 3. 졸릴 때는 뇌가 아니라 몸으로

- 뇌는 뇌로 이기지 못한다. **몸으로 이긴다.**
- 졸릴 때 억지로 깨려 하지 말고 **손·몸 전체를 움직여** 리프레시한다.

## 4. 집중 시간은 훈련이다

- 집중 시간도 훈련으로 늘린다: **10분 → 20분 → 목표 50분.**
- **타이머로 시각화**한다.
- **아주 짧은 목표**를 설정한다. (30분 공부에도 "단어 20개", "몇 페이지" 같은 목표)
  - 잘 모를 땐 시간을 목표로 삼는 것도 괜찮다.
