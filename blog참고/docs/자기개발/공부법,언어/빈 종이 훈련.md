# 빈 종이 훈련

## 무엇인가

어떤 내용이든 **내가 배운 걸 빈 종이에 출력**하는 방식이다.

- **내가 이해한 것**을 출력한다. (내용을 그대로 베끼기 X — 가끔은 그대로도 OK)
- 텍스트만으로 출력하기보다 **이미지(그래프·도식·생각나무 등)** 도 함께하면 더 좋다.

## 왜 효과적인가

- **가르칠 수 있을 정도**로 출력 → 위 과정을 반복하는 것이 곧 복습이다.
- 책 한 권을 읽어도 **읽은 데서 끝내지 말고**, 요약이든 다른 생각이든 반드시 출력한다.

> 핵심: 입력(읽기)에서 멈추지 말고 **출력(백지 쓰기)** 까지 가야 진짜 내 것이 된다.
