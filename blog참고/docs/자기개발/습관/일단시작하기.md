# 미루는 습관, 일단 시작하는 법

## 1. 트리거(Trigger) 만들기

- 시작을 부르는 **트리거**를 정한다. (예: 졸릴 때 hot sauce, 일어날 때 hot sauce)
- **트리거는 변하지 않는 것**이 좋다.
- 일할 때 **조용한 음악** — 일에 집중되어 음악이 방해되면 그때 끈다.
- 아니면 **펜 돌리기** 같은 작은 동작.

## 2. 어떻게 시작할까 = 작게 나눠서 일단

- **일단 시작**한다. 작게 나눠서 시작한다.
- 분명히 마음 어딘가에는 **지금처럼 살고 싶어 하는** 관성이 있다. 그걸 이겨낸다.

## 3. 하루를 승리하는 법 — 처음 1시간

- 하루를 이기고 싶으면 **처음 1시간이 중요**하다.
- 뭐든 **일을 다 끝내고 노는 게 최고.**
- **To-do list와 Not-to-do list 둘 다** 좋다.
  - **Not-to-do**: To-do를 끝내기 전까지는 안 하는 것. (끝냈다면 해도 좋다)
  - 하루 할당량을 채운 다음엔 **노는 게 최고.**
- 인뽑(인스타그램 등) 금지.

## 4. 지원·사이드 프로젝트도 "일단"

- 사이드 프로젝트도 **일단** 시작한다.
- **지원도 마찬가지** — 기술 스택을 안 다뤄봤어도 **일단 지원**한다.
